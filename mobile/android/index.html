ololo-relay для Android

Версия v0.8.10

Скачать

  • OloloRelay-v0.8.10.apk — APK, ~51 MB
  • SHA256SUMS — контрольная сумма (sha256sum -c SHA256SUMS).
  • latest.txt — текущая версия (машиночитаемо).

Установка

  1. Скачать APK на устройство (через браузер либо adb install).
  2. В системных настройках разрешить установку из неизвестных источников для браузера (Android ≥ 8) либо подтвердить запрос при установке.
  3. Открыть скачанный APK, подтвердить.

Особенности

  • Сборка подписана self-signed release-ключом — обновления накатываются поверх установленной без удаления. На Play Store такая подпись не зайдёт, для прямой раздачи через этот сайт достаточно.
  • Приложение — одновременно VPN-клиент mesh и админка для управления кластером (всегда под рукой на телефоне).
