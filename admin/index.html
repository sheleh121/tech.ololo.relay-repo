ololo-relay-admin

Десктоп-приложение для управления mesh ololo-relay (peers, routes, policy, capture).

Платформы

  • Windows x64 — installer и portable
  • Linux x64 — через пакетный репозиторий: apt install ololo-relay-admin / dnf install ololo-relay-admin (подключение репозитория — setup.sh)
