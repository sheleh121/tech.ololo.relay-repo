repo.ololo.tech

Пакетный репозиторий продуктов ololo.tech.

Что здесь

  • /apt — APT-репозиторий для Debian/Ubuntu (amd64, stable).
  • /rpm — DNF/YUM-репозиторий для Fedora и RedOS (x86_64).
  • /admin — десктоп-админка для управления mesh: win-x64.
  • /pubkey.asc — публичный GPG-ключ репозитория.
  • /setup.sh — автоматический bootstrap.

Быстрая установка

curl -fsSL https://repo.ololo.tech/setup.sh | sudo sh

Скрипт определяет дистрибутив, импортирует GPG-ключ и подключает репозиторий. После него:

sudo apt install ololo-relay   # Debian/Ubuntu
sudo dnf install ololo-relay   # Fedora / RedOS

Продукты

  • ololo-relay — relay-узел. Описание и гайд.
  • ololo-relay-admin — десктоп-приложение для управления mesh. Скачать: Windows x64 (установщик и portable, SHA256SUMS рядом). Linux/macOS — в планах.

Ручная настройка

Если не хочется запускать чужой sh через curl — см. полные инструкции и проверку отпечатка ключа на странице продукта.
